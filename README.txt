Алгорим работы скрипта:

1) Cчитывает CSV файл
2) Передаёт отзывы в chatGPT
3) В ответ получает райтинг отзыва и настроение автора
4) Записывает результат в 'reviews\\data_analyzed.csv'

Для оптимизации траты токенов оба задания для каждого отзыва выполняются в одном общем запросе к GPT.

Вход в программу через 'moduls\\main.py'
